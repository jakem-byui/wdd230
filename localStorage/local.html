What is Local Storage?

  • Local Storage is a web storage solution provided by web browsers to store data persistently on a user's device. Unlike session storage, local storage retains data even when the browser is closed and reopened.
  • Local Storage is similar to sessionStorage, except that while localStorage data has no expiration time, sessionStorage data gets cleared when the page session ends â€” that is, when the page is closed. (localStorage data for a document loaded in a "private browsing" or "incognito" session is cleared when the last "private" tab is closed.)
    Reference: localStorage

How do I access Local Storage?
Local storage is accessible in any window or tab that is open to your site. This means if you store some data in local storage on one tab of your browser that same local storage data will be available on all other tabs and windows you have open to that site.

// Storing data
localStorage.setItem('username', 'JohnDoe');

// Retrieving data
const username = localStorage.getItem('username');
console.log(username); // Outputs: JohnDoe

How long does Local Storage last?
Once you store data in local storage it will stay there until you manually remove it or the user manually removes it.
